◆ khc_sock_code_t

enum khc_sock_code_t

Indicates socket callback result.

Enumerator
KHC_SOCK_OK 

Retrun this code when operation succeed.

KHC_SOCK_FAIL 

Return this code when operation failed.

KHC_SOCK_AGAIN 

Return this code when operation is in progress.

This code is used when the underlying socket is non-bloking mode. khc calls the callback again with the same argument until the callbacks returns KHC_SOCK_OK or KHC_SOCK_FAIL.
