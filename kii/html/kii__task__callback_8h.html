◆ KII_CB_TASK_CONTINUE

typedef kii_bool_t(* KII_CB_TASK_CONTINUE) (void *task_info, void *userdata)

Callback determines whether to continue task execution.

If discontinued, KII_CB_TASK_EXIT callback is called.

Parameters
[in]task_infoInformation of task. The type of this parameter is defined by the module requires task execution.
[in]userdataContext object pointer.
Returns
KII_TRUE continues task execution, returning Kii_FALSE results task exit.
